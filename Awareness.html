Cyber Security Awareness

Learn how to protect yourself online with simple, effective cyber safety practices. Each topic explains how attackers target you and how you can stay safe.

1. System Security

Keep your system updated, use antivirus, avoid pirated software, and never install unknown apps. Updates patch vulnerabilities before attackers exploit them.

2. Detecting Viruses

Slow device? Pop-ups? Unknown apps? High CPU usage? These are virus signs. Use trusted antivirus, run a full scan, and remove suspicious programs immediately.

3. Phishing Safety

Avoid clicking unknown links, double-check email senders, never share passwords, and always verify websites before entering personal info.

4. Password Protection

Use strong, unique passwords. Combine letters, numbers, symbols. Enable 2-factor authentication for extra security.

5. Network Safety

Avoid public Wi-Fi for banking or logins. Use VPN for secure browsing. Secure your Wi-Fi with a strong password.
