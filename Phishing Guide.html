Phishing Attack Guide

Phishing is an attempt to trick you into revealing credentials, financial details or installing malware using deceptive messages (email, SMS, social media).

Common Indicators

  • Slightly altered sender addresses
  • Urgent and threatening tone
  • Mismatched link destinations
  • Requests for sensitive data/OTPs

How to Verify

  • Hover links (don't click!)
  • Type domains manually
  • Contact sender via official channels

Protective Measures

  • Enable Multi-Factor Authentication (MFA)
  • Use email filters and anti-phishing tools
  • Never share OTPs on external links
  • Report suspicious activity immediately
