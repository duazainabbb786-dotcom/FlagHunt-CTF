Social Media Security Guide

Social platforms are useful but oversharing risks privacy, reputation and targeted attacks.

Privacy Habits

  • Set accounts to private
  • Limit post visibility
  • Review app permissions
  • Think before you post

Security & Recovery

  • Enable 2FA (MFA)
  • Verify friend requests
  • Report impersonation
  • Use recovery options
